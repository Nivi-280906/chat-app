# 💬 Real-Time Chat Application

A real-time chat application built using **Node.js, Express, and Socket.IO** that enables instant communication between users.

---

## 🚀 Live Demo

👉 https://chat-app-pht4.onrender.com

---

## 📊 Monitoring

👉 https://chat-app-pht4.onrender.com/metrics

---

## ✨ Features

* ⚡ Real-time messaging using Socket.
* 🎨 Clean and responsive UI
* 🔄 CI/CD pipeline using GitHub Actions
* 🐳 Docker support
* 📈 Monitoring using Prometheus metrics

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Real-time: Socket.IO
* CI/CD: GitHub Actions
* Monitoring: Prometheus (prom-client)

---

## ⚙️ How to Run Locally

```bash
npm install
npm start
```

Then open:
http://localhost:5000

## 📌 Deployment
